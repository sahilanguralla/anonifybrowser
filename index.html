Anonify Browser

Anonify Browser lets you modify your browsers identity to another browser thereby hiding your browser from websites. This add-on might help you in situations where a website forces you to use a specific browser or operating system. There are about 100 OS versions/browser and about 7 Operating Systems.

How will it help ?

Anonify Browser is intended to use for modifying your browser identity, technically modifying the User Agent string. But it not only hides your browser from websites but also :

  • Allows you to switch to a browser that a website recommends
  • Decreases Page Load on switching to a Smartphone User Agent.
  • Allows downloading of files that requires you to be using a specific OS.
  • Much more uses you can discover

Principle behind Anonify Browser

If you are thinking about what is really happening at back-end you might be amazed to know it is not as complex as it appears. It actually plays with a Mozilla preference - general.useragent.override. Whenever you select a browser, the preference is updated to corresponding user agent string

Coming Updates

Future versions may include following features :

  • notifications about currently active user agent
  • Update button to update user agent list
